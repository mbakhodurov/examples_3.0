# Базовый тест на стандартном testing

Минимальный пример: тестируемая функция `isStupid()` и два подхода к её проверке.

## На что обратить внимание

| Что | Файл | Зачем |
|-----|------|-------|
| Антипаттерн: «ручной» тест | `main.go` (`testIsStupid()`) | Функция-тест без интеграции с `go test` — нигде не вызывается, нет ассертов, не запустить из CLI |
| Правильный тест | `main_test.go` | `testing.T` + `t.Error()` — минимальный рабочий тест, запускается через `go test` |

Здесь намеренно нет testify, table-driven, моков — только голый `testing.T` с ручными `if`-проверками.

## Как запустить

```bash
go test ./...
```
